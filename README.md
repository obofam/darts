# 🎯 ダーツの旅

日本全国の市区町村からランダムに行き先を決めるWebアプリ。  
所さんの「ダーツの旅」を再現！人口が多いところほど当たりやすい重み付き抽選。

## ✨ 特徴

- **全国1,199市区町村**を収録（47都道府県すべてカバー）
- **人口加重抽選** — Google Mapsの情報量が多い都市ほど当たりやすい
- **ルーレット演出** — スロットマシン風の減速アニメーション
- **Google Maps連携** — 結果をワンタップでGoogle Mapsで確認

## 🚀 使い方

1. [**こちらから遊ぶ →**](https://obofam.github.io/darts/)
2. 「投げる！」ボタンを押す
3. ルーレットが回って…今日の目的地が決定！

## 🛠️ 技術

- **Vanilla HTML / CSS / JavaScript** — フレームワーク不要
- 人口データ: 令和2年国勢調査ベース概数
- レスポンシブ対応（スマホ・タブレット・PC）

## 📁 ファイル構成

```
├── index.html          # メインページ
├── styles.css          # スタイリング（ダークテーマ）
├── app.js              # アプリロジック（抽選・演出）
└── municipalities.js   # 市区町村データ（人口付き）
```
